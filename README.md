## ✨ Features

- 🎬 **60+ CSS Animations** — floating elements, scroll reveals, particle bursts, counter animations
- 🛒 **Product Catalog** — 9 dairy products with filter tags and hover interactions
- ⭐ **Reviews Section** — real customer reviews with star ratings
- 📊 **Popular Times Chart** — interactive bar chart showing busy hours per day
- 📬 **Contact Form** — with validation and animated success state
- 🏃 **Marquee Ticker** — animated scrolling product banner
- 📱 **Fully Responsive** — works on mobile, tablet and desktop
- ⚡ **Blazing Fast** — built with Vite for instant hot reload

## 🎨 Animations Used

| Animation | Where |
|-----------|-------|
| `float` | Hero emoji, footer, stat icons |
| `heroText` | Hero headline entrance |
| `borderDance` | Morphing background blobs |
| `marquee` | Product ticker strip |
| `fadeUp / slideIn` | Scroll reveal on all sections |
| `countUp` | Stats counter (110+, 9+, 4.4★) |
| `glowPulse` | Call Now button glow |
| `particleDrift` | Click particle burst in hero |
| `wobble` | Logo milk emoji |
| `tagFloat` | Product badge tags |
